# Roadmap

## Fase 1: Base funcional

- Projeto Django criado.
- Modelos iniciais de dominio.
- Admin configurado.
- Documentacao dos fluxos principais.
- Painel inicial simples.

## Fase 2: Operacao manual assistida

- Criar dados reais pelo admin.
- Ajustar status e campos conforme uso.
- Adicionar filtros e acoes administrativas.
- Criar comandos Django para importar e-mails de arquivos ou mocks.

## Fase 3: Integracao com e-mail

- Implementar OAuth para Gmail ou Outlook.
- Buscar mensagens por remetentes configurados.
- Registrar e-mails relevantes.
- Classificar e sugerir status automaticamente.

## Fase 4: Notificacoes e calendario

- Criar rotina de lembretes pendentes.
- Enviar notificacoes por e-mail.
- Integrar calendario externo.

## Fase 5: Preenchimento assistido

- Mapear campos comuns de formularios.
- Gerar sugestoes a partir do perfil.
- Validar formato de dados por tipo de campo.
- Avaliar extensao de navegador para inserir sugestoes no site externo.

## Fase 6: API e frontend

- Adicionar Django REST Framework.
- Construir frontend React.
- Melhorar dashboard, filtros e fluxos de revisao.
