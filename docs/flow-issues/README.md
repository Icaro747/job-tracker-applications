# Problemas de fluxo

Esta pasta registra problemas de experiencia, modelagem ou regra de negocio que
aparecem durante o uso real do produto e que precisam de analise posterior.

Use estes documentos para separar diagnosticos de fluxo das specs formais. Um
item aqui nao e necessariamente uma decisao de produto; ele descreve o problema,
as evidencias observadas, o impacto e as perguntas abertas antes de virar uma
spec ou uma tarefa de implementacao.
